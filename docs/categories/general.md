# 🔧 Général

- [`§bonjour`](../commands/bonjour.md) — Le bot vous dit bonjour.
- [`§help`](../commands/help.md) — Aide générale, ou détail d'une commande : help [commande].
- [`§ping`](../commands/ping.md) — Affiche la latence du bot.
- [`§status`](../commands/status.md) — Version, ping et nombre de serveurs.
- [`§uptime`](../commands/uptime.md) — Affiche depuis combien de temps le bot tourne.
