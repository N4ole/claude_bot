# 👑 Owner du bot

- [`§addowner`](../commands/addowner.md) — Ajoute un owner au bot.
- [`§central`](../commands/central.md) — Tableau de bord global du bot (owners).
- [`§helpowner`](../commands/helpowner.md) — Pas de description.
- [`§invite`](../commands/invite.md) — Génère une invitation vers un serveur du bot.
- [`§owners`](../commands/owners.md) — Liste les owners du bot.
- [`§reload`](../commands/reload.md) — Recharge un cog à chaud (ou 'all' pour tout recharger).
- [`§respond`](../commands/respond.md) — Répond en MP à un propriétaire de serveur, ou `all` pour une annonce.
- [`§rmowner`](../commands/rmowner.md) — Retire un owner du bot.
- [`§say`](../commands/say.md) — Fait parler le bot dans le salon courant.
- [`§serveurs`](../commands/serveurs.md) — Détaille chaque serveur du bot (une page par serveur).
- [`§shutdown`](../commands/shutdown.md) — Éteint le bot.
