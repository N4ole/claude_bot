# 👑 Owner

- [`§addowner`](../commands/addowner.md) — Ajoute un owner au bot. — 🔒 *Owner du bot*
- [`§helpowner`](../commands/helpowner.md) — Pas de description. — 🔒 *Owner du bot*
- [`§invite`](../commands/invite.md) — Génère une invitation vers un serveur du bot. — 🔒 *Owner du bot*
- [`§owners`](../commands/owners.md) — Liste les owners du bot. — 🔒 *Owner du bot*
- [`§reload`](../commands/reload.md) — Recharge un cog à chaud (ou 'all' pour tout recharger). — 🔒 *Owner du bot*
- [`§rmowner`](../commands/rmowner.md) — Retire un owner du bot. — 🔒 *Owner du bot*
- [`§say`](../commands/say.md) — Fait parler le bot dans le salon courant. — 🔒 *Owner du bot*
- [`§serveurs`](../commands/serveurs.md) — Détaille chaque serveur du bot (une page par serveur). — 🔒 *Owner du bot*
- [`§shutdown`](../commands/shutdown.md) — Éteint le bot. — 🔒 *Owner du bot*
